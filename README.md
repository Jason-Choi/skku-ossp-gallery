# SKKU OSSP Gallery

Project gallery for open-source software lecture at Sungkyunkwan University.

[Link](https://git.jasonchoi.dev/skku-ossp-gallery)

## Development
Installation
```
yarn
```
Development (Dev server with Next.js)
```
yarn dev
```
Build and Deploy (to Github Pages)
```
yarn deploy
```
